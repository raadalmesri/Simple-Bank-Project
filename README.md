# Bank Management System (C++)

A console-based banking management system developed using C++.

## Features

- Add new clients
- Show all clients
- Find client
- Update client
- Delete client
- Deposit
- Withdraw
- Show total balances

## Technologies

- C++
- File Handling
- Struct
- Vector
- Functions
- Enumerations

## Data Storage

The application stores client data in a text file.
